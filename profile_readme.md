## 🚀 Featured Project
### 📦 [SAP Custom Inventory Management & Replenishment](https://github.com/aaryan4747/sap-custom-inventory-management)
An end-to-end custom inventory automation system built on SAP NetWeaver AS ABAP.
* **Database**: Custom transparent tables (`ZMINVENT_MASTER` / `ZMINVENT_MOVE`) with strict validation.
* **Logic**: OO-ABAP balance calculator class dynamically updating stocks via delta logs.
* **Automation**: Standard `MB_DOCUMENT_BADI` hook interceptor to auto-trigger purchase requisitions.
* **Reporting**: `CL_SALV_TABLE` interactive ALV Grid report with colored alerts and double-click drilldown history.
* **Simulator**: A premium web-based Fiori playground simulating the complete SAP server/client integration.
