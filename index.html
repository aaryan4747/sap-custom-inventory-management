Interactive Simulation Walkthrough

This dashboard simulates an end-to-end integration of SAP custom warehouse tracking components. Follow these steps to witness the automation in action:

1
Inspect Initial Stock

Go to the ALV Grid Report tab. Observe the current stock levels, safety thresholds, and color indicators.

2
Post Goods Movement

Navigate to MIGO Movement. Post a Goods Issue (201) to reduce stock below the safety threshold.

3
Observe BAdI Automation

Watch the BAdI logs run in real-time. The BAdI intercepts the movement, checks thresholds, and triggers the replenishment workflow.

4
Verify ALV Grid & PR

Return to the ALV Grid to view updated levels, red warnings, and the auto-generated Purchase Requisition (PR) document.
